# 图片压缩工具

批量压缩图片文件大小，同时尽量保持图片质量。

## 功能特点

- **批量压缩**：支持同时处理多张图片
- **质量控制**：可调节压缩质量（1-100）
- **尺寸限制**：可设置压缩后图片的最大宽度
- **格式选择**：支持输出为PNG、JPEG或WebP格式
- **实时预览**：显示压缩前后的文件大小对比
- **单文件下载**：可单独下载每张压缩后的图片

## 支持格式

- JPG / JPEG
- PNG
- WebP
- GIF

## 使用方法

1. 点击"选择图片"按钮或拖拽图片文件到上传区域
2. 支持选择多张图片
3. 调整压缩参数：
   - 设置输出格式（PNG/JPEG/WebP）
   - 调整图片质量滑块
   - 设置最大宽度（可选）
4. 点击"开始压缩"按钮
5. 压缩完成后可单独或批量下载压缩后的图片

## 技术实现

- 使用 Canvas API 进行图片压缩处理
- 纯前端实现，文件处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸

## 隐私说明

本工具完全在您的浏览器本地运行，所有图片数据**不会**上传到任何服务器。您的隐私得到完全保护。
