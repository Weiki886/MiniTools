# 图片取色器

上传图片后，点击图片任意位置即可获取该位置的颜色代码（支持HEX、RGB、HSL格式）。

## 功能特点

- **点击取色**：点击图片任意位置获取精确颜色
- **多种格式**：同时显示HEX、RGB、HSL三种颜色代码
- **一键复制**：点击颜色代码或复制按钮快速复制到剪贴板
- **坐标显示**：实时显示当前鼠标位置的坐标
- **实时预览**：颜色预览区域实时显示当前选中的颜色

## 支持格式

- JPG / JPEG
- PNG
- GIF
- BMP
- WebP

## 使用方法

1. 点击"选择图片文件"按钮上传图片
2. 图片加载后，点击图片上的任意位置
3. 右侧会显示该位置的颜色信息
4. 点击颜色代码或"复制"按钮复制到剪贴板

## 技术实现

- 使用 Canvas API 获取图片像素颜色数据
- 纯前端实现，图片处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸

## 隐私说明

本工具完全在您的浏览器本地运行，所有图片数据**不会**上传到任何服务器。您的隐私得到完全保护。
