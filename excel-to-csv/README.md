# Excel转CSV工具

快速将您的Excel文件（.xlsx、.xls）转换为CSV格式。

## 功能特点

- **批量转换**：支持同时处理多个Excel文件
- **数据预览**：转换前可预览数据内容，确保数据正确
- **编码支持**：支持UTF-8编码，兼容各种语言字符
- **格式保留**：尽量保留原始数据的格式和结构

## 支持格式

- .xlsx（Excel 2007+）
- .xls（Excel 97-2003）

## 使用方法

1. 点击"选择文件"按钮或拖拽Excel文件到上传区域
2. 支持选择多个文件
3. 点击"预览数据"查看文件内容
4. 检查数据是否正确读取
5. 点击"确认转换"开始转换
6. 转换完成后可单独或批量下载CSV文件

## 技术实现

- 使用 [SheetJS (xlsx)](https://github.com/sheetjs/sheetjs) 库解析Excel文件
- 纯前端实现，文件处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸
