# 图片质量增强

在线调整图片的各项参数，改善图片质量。

## 功能特点

- **亮度调节**：调整图片的整体明暗程度（-100到+100）
- **对比度调节**：调整图片的明暗对比（-100到+100）
- **锐化**：增强图片的边缘清晰度
- **饱和度调节**：调整图片的色彩饱和程度（-100到+100）
- **色温调节**：调整图片的冷暖色调（-100到+100）
- **曝光度调节**：调整图片的整体曝光量（-100到+100）
- **伽马校正**：调整图片的伽马曲线（0.10到3.00）
- **噪点减少**：减少图片中的噪点
- **色调调节**：调整图片的整体色调（-180到+180）
- **实时预览**：左右对比原图和调整后的效果
- **多格式下载**：支持导出为PNG、JPEG或WebP格式

## 使用方法

1. 点击上传区域或拖拽图片文件上传图片
2. 在左侧调整面板中调整各项参数
3. 点击"应用调整"按钮查看效果
4. 在右侧预览区域对比原图和调整后的效果
5. 设置文件名并选择输出格式
6. 点击"下载调整后的图片"获取结果

## 技术实现

- 使用 Canvas API 进行图片像素处理
- 纯前端实现，图片处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸
