# 音频调整器

在线调整音频文件的各种参数，包括音量、播放速度，并支持添加淡入淡出效果。

## 功能特点

- **音量调整**：调节音频的音量大小（0% - 200%）
- **播放速度**：调整音频的播放速度（0.5x - 2.0x）
- **淡入效果**：为音频开头添加渐入效果
- **淡出效果**：为音频结尾添加渐出效果
- **实时预览**：调整参数后可直接试听效果
- **批量处理**：支持处理单个或多个音频文件

## 支持格式

- MP3
- WAV
- OGG
- FLAC
- M4A

## 使用方法

1. 点击"选择音频文件"按钮，或将音频文件拖拽到上传区域
2. 选择或拖拽音频文件后，在右侧面板中调整各项参数
3. 点击"预览"按钮试听调整后的效果
4. 满意后点击"下载"按钮获取处理后的音频文件

## 技术实现

- 使用 Web Audio API 进行音频处理
- 纯前端实现，音频处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸

## 隐私说明

本工具完全在您的浏览器本地运行，所有音频数据**不会**上传到任何服务器。您的隐私得到完全保护。
