# PNG转ICO转换器

快速将PNG图片转换为Windows图标文件（ICO格式）。

## 功能特点

- **多种尺寸**：支持生成16x16到256x256等多种尺寸的图标
- **高质量转换**：保证图标在各种尺寸下的清晰度
- **实时预览**：转换前可预览PNG图片效果
- **简单易用**：一键完成转换和下载

## 支持尺寸

- 16x16（适用于小图标）
- 32x32（标准图标）
- 48x48（大图标）
- 64x64（特大图标）
- 128x128（超大图标）
- 256x256（高清图标）

## 使用方法

1. 点击"选择PNG图片"按钮或拖拽PNG文件到上传区域
2. 选择所需的ICO图标尺寸
3. 点击"转换为ICO"按钮开始转换
4. 转换完成后点击"下载ICO文件"按钮获取文件

## 建议

- 上传尺寸为256x256像素或更大的PNG图片以获得最佳效果
- 确保PNG图片背景透明（如果需要透明背景的图标）

## 技术实现

- 纯前端实现，使用Canvas API进行图片处理
- 文件处理完全在浏览器本地进行，无需服务器
- 响应式设计，支持各种屏幕尺寸

## 隐私说明

本工具完全在您的浏览器本地运行，所有图片数据**不会**上传到任何服务器。您的隐私得到完全保护。
