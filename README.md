# MiniTools

一个实用的在线工具集合，无需安装，直接在浏览器中使用。所有工具完全免费，隐私安全有保障。

## 工具列表


| 工具 | 说明 |
| ------------------------------------------------------------- | ---------------------------- |
| [audio-adjuster](./audio-adjuster/) | 音频调整器 - 调整音频的音量、速度、添加淡入淡出效果 |
| [color-picker](./color-picker/) | 图片取色器 - 上传图片点击任意位置获取颜色代码 |
| [excel-to-csv](./excel-to-csv/) | Excel转CSV - 将Excel文件转换为CSV格式 |
| [ico-icon-conversion](./ico-icon-conversion/) | PNG转ICO - 将PNG图片转换为ICO图标文件 |
| [image-compressor](./image-compressor/) | 图片压缩 - 批量压缩图片文件大小 |
| [image-to-pdf](./image-to-pdf/) | 图片转PDF - 将多张图片合并为PDF文档 |
| [large-number-calculator](./large-number-calculator/) | 大数计算器 - 支持大数字的加减乘除计算 |
| [pdf-to-image](./pdf-to-image/) | PDF转图片 - 将PDF转换为图片格式 |
| [picture-quality-enhancement](./picture-quality-enhancement/) | 图片质量增强 - 调整图片的亮度、对比度、锐化等参数 |
| [text-to-image](./text-to-image/) | 文字转图片 - 将文字内容生成为图片 |
| [video-to-audio](./video-to-audio/) | 视频转音频 - 从视频中提取音频 |


## 隐私承诺

所有工具均采用纯前端技术实现，处理过程完全在您的浏览器本地进行。**您的数据不会上传到任何服务器**，隐私安全有保障。

## 技术栈

- HTML5 / CSS3 / JavaScript
- 前端框架与库：
  - [jsPDF](https://github.com/MrRio/jsPDF) - PDF生成
  - [pdf.js](https://github.com/mozilla/pdf.js) - PDF解析
  - [SheetJS](https://github.com/sheetjs/sheetjs) - Excel处理
  - [FFmpeg.wasm](https://github.com/ffmpegwasm/ffmpeg.wasm) - 音视频处理
