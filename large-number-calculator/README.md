# 大数计算器

支持超大数字的加减乘除计算器，突破JavaScript数字精度限制。

## 功能特点

- **大数支持**：可计算任意大小的整数和浮点数
- **四则运算**：支持加法、减法、乘法、除法运算
- **高精度**：保留计算结果的完整精度，无科学计数法
- **实时计算**：输入数字后立即显示计算结果
- **清除功能**：一键清除所有输入

## 使用方法

1. 输入第一个数字
2. 选择运算符号（+、-、×、÷）
3. 输入第二个数字
4. 点击"计算"按钮或按回车键获取结果
5. 如需继续计算，可点击"使用结果"将当前结果作为第一个数字

## 技术实现

- 使用字符串处理算法实现大数运算
- 纯前端实现，计算过程完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸
