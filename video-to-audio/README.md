# 视频转音频工具

从视频文件中提取音频，支持转换为MP3、WAV等常见音频格式。

## 功能特点

- **格式多样**：支持从各种视频格式提取音频
- **多格式输出**：支持导出为MP3、WAV、OGG等格式
- **质量选择**：可选择音频的质量和比特率
- **实时预览**：转换前可预览音频效果
- **批量处理**：支持处理多个视频文件

## 支持输入格式

- MP4
- WebM
- AVI
- MOV
- MKV

## 输出格式

- MP3
- WAV
- OGG

## 使用方法

1. 点击"选择视频文件"按钮或拖拽视频文件到上传区域
2. 支持选择多个视频文件
3. 在设置面板中选择输出音频格式
4. 选择音频质量/比特率（可选）
5. 点击"开始转换"按钮
6. 转换完成后可单独或批量下载音频文件

## 技术实现

- 使用 [FFmpeg.wasm](https://github.com/ffmpegwasm/ffmpeg.wasm) 进行音视频处理
- 纯前端实现，文件处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸
