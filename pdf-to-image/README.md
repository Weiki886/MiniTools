# PDF转图片工具

将PDF文件转换为图片格式，支持分页图片输出或长图合成。

## 功能特点

- **多种模式**：支持分页图片和长图合成两种输出模式
- **格式选择**：支持输出为PNG或JPEG格式
- **质量调节**：可调节输出图片的质量倍数（0.1x - 2.0x）
- **批量处理**：一次性转换PDF的所有页面
- **实时预览**：转换后可预览生成的图片

## 使用方法

1. 点击"选择PDF文件"按钮或拖拽PDF文件到上传区域
2. 在转换设置中选择输出模式：
   - **分页图片**：每页PDF生成一张独立图片
   - **长图合成**：将所有页面合并为一张长图
3. 选择输出图片格式（PNG/JPEG）
4. 调整图片质量（可选）
5. 点击"开始转换"按钮
6. 转换完成后点击"下载图片"获取结果

## 技术实现

- 使用 [pdf.js](https://github.com/mozilla/pdf.js) 库解析PDF文档
- 使用 Canvas API 渲染PDF页面为图片
- 纯前端实现，文件处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸
