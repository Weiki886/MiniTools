# 文字转图片工具

将文字内容生成为图片，支持自定义字体、颜色和背景。

## 功能特点

- **字体大小**：可调节文字大小（12px到72px）
- **多种字体**：支持Arial、Times New Roman、Courier New、宋体、微软雅黑等多种字体
- **文字颜色**：可选择任意文字颜色
- **背景颜色**：可选择任意背景颜色
- **实时预览**：生成的图片可实时预览
- **格式下载**：支持下载为PNG格式的图片

## 使用方法

1. 在文本框中输入要转换的文字内容
2. 调整字体大小滑块设置合适的字号
3. 从下拉菜单中选择字体样式
4. 分别设置文字颜色和背景颜色
5. 点击"生成图片"按钮生成预览
6. 点击"下载图片"按钮下载生成的图片

## 技术实现

- 使用 Canvas API 绘制文字和背景
- 纯前端实现，图片处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸

## 隐私说明

本工具完全在您的浏览器本地运行，所有文字数据**不会**上传到任何服务器。您的隐私得到完全保护。
