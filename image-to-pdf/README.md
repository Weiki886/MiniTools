# 图片转PDF工具

将多张图片合并为单个PDF文档的在线工具。

## 功能特点

- 支持批量选择多张图片（JPG、PNG、GIF等格式）
- 可自定义PDF文件名
- 提供"优化图片清晰度"选项，可选择生成更高质量但文件更大的PDF
- 实时预览已选择的图片
- 转换完成后一键下载生成的PDF文件

## 使用方法

1. 点击"选择图片"按钮或拖拽图片文件到上传区域
2. 可选：输入自定义的PDF文件名（默认为 `converted_document`）
3. 选择是否勾选"优化图片清晰度"选项
4. 点击"转换并下载PDF"按钮开始转换
5. 等待转换完成后，PDF文件会自动下载到本地

## 技术实现

- 使用 [jsPDF](https://github.com/MrRio/jsPDF) 库生成PDF文档
- 纯前端实现，图片处理完全在浏览器本地进行
- 响应式设计，支持各种屏幕尺寸

## 隐私说明

本工具完全在您的浏览器本地运行，所有图片数据**不会**上传到任何服务器。您的隐私得到完全保护。
